# Design Context - Atlas Balance AI Chat

## Users
CFOs and treasury managers of small-to-medium enterprises working in local network. They use the AI chat for quick financial insights: account spending analysis, anomaly detection, trend questions. Usage is opportunistic—between statement reviews, during dashboard analysis. They need reliable, precise answers they can trust.

## Brand Personality
**Precise, Serious, Premium**

Tone: Like a trusted auditor or financial analyst. Direct, no fluff. Clinical but not cold. Builds confidence through competence and clarity.

## Aesthetic Direction
- **Reference**: Intercom-style chat widget (compact, focused, minimal)
- **Theme**: Coherent with existing Atlas Balance UI (National Park + Hind Madurai fonts, #285bd9 primary blue, light/dark modes via CSS variables)
- **Positioning**: Bottom-right corner (like Intercom, Drift, etc.)
- **Anti-references**: Avoid playful, colorful, or "chat bubble" aesthetics. No rainbow accents, no mascots, no casual tone in visuals.

## Design Principles
1. **Precision over personality**: Every pixel serves the financial data, not decoration
2. **Premium through restraint**: Luxury is in refinement, not ornamentation. Clean typography, breathing room, intentional spacing
3. **Auditor-grade trust**: Dark surfaces, high contrast, monospace for numbers, no visual tricks that distract
4. **Seamless integration**: Match existing app variables, spacing scale, animation easing—feels like a native component
5. **Compact utility**: Optimized for quick Q&A, not long conversations. Get insight, move on.

## Implementation Details
- Position: fixed bottom-right (with safe area insets for mobile)
- Size: ~380px width (compact but readable)
- Colors: Reuse existing palette (--bg-surface, --accent-primary, --text-primary, etc.)
- Fonts: "National Park" for headers, "Hind Madurai" for body, "Atlas Mono" for data
- Motion: Smooth slide-in from bottom-right, no bounce or elastic easing
- Dark mode: Full support via [data-theme="dark"] and CSS variables
